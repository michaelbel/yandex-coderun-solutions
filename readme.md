# Yandex CodeRun Solutions

Решения задач с платформы [Yandex CodeRun](https://coderun.yandex.ru) — сервиса для подготовки к техническим собеседованиям и олимпиадам по программированию от Яндекса.

Решения написаны на **Kotlin** и **Java**.

| [Впечатляющие тренировки](https://coderun.yandex.ru/selections/mgustokashin) — сложные задачи с Тренировок по алгоритмам Яндекса от M. Gustokashin | [mgustokashin/](mgustokashin/readme.md) |
